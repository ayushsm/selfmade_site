Quant Trading

Wall Street 2.0

An introduction to algorithm-based stock trading. Beginning with the Ichimoku Cloud and fundamental trading theories, I learned the basics behind most algorithms used in micro-transactions and day-to-day trading algorithms. From there, I was able to create my own algorithm, resulting in a near $10K net gain using past years' analysis. Eventually, I hope to be able to refine the algorithm to be put into real-time application.

UI/UX Dev

The customer is king

There aren't many opportunities to learn design in on campus and design is so pivotal to the CS industry, so I thought teaching about it would be useful to many people. There aren't many technologies used in the project, but the repo details the project's creation as well as what the teams did with the project.
